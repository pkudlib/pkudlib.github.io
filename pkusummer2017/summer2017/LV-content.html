Syllabus

  • Propositional Logic:
    • Syntax, semantics, substitution, normal forms, equivalences
    • Satisfiability, getting familiar with a SAT-solver, application in constraint solving and verification
    • Decision procedures for satisfiability
  • First-Order Logic:
    • Syntax, semantics, substitution, normal forms, equivalences
    • Satisfiability, model checking, undecidability
  • Temporal Logic (Linear Temporal Logic and Computation Tree Logic):
    • Syntax, semantics, modelling systems
    • Model checking algorithms, getting familiar with a model checker, application in verification
  • First-Order Theories and Program Verification:
    • Satisfiability Modulo Theories (SMT) and SMT-solvers
    • Theory of Linear Arithmetic
    • Operational Semantics, Hoare Logic, and Application to Program Verification
  • Student Presentations

教学方式

课堂授课为主，课后读论文、做调研及汇报为辅。


学生成绩评定办法

Presentation（10%），Project（40%），Final Take-Home Exam（50％）
